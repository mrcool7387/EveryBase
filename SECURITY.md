# Security Policy

## Reporting a Vulnerability
We take the security of EveryBase seriously. If you find a security vulnerability, 
please **do not** open a public issue. Instead, please report it privately.

### Contact
Please send your report to: **alexanderschwarz148@gmail.com**

### Process
1. Once a report is received, we will acknowledge as soon as possible.
2. We will work on a fix as quickly as possible.
3. We will notify you once the vulnerability is patched.

### Rewards
Outstanding security fixes may be eligible for voluntary compensation 
under the EveryBase Public License (EBPL) Section 3.4.

Thank you for helping keep EveryBase secure!
